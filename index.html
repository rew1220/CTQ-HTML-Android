• html, head, body 테그
    • html테그는 파일의 head와 body를 감쌈
    • head테그는 이 부분이 본문 이외의 것이라는것을 알림(title, meta 등등..)
    • body 테그는 이 부분이 본문임을 설명함
  • title 테그
    • 페이지의 제목을 나타냄 즉, 본문을 설명함
    줄바꿈 = 자동x

    줄바꿈 키워드 br은 닫지않음(/br이 없음)

    p는 그루핑 하나의 단락을 그룹화시킴 이때는 닫는 기호를 사용함(p~/p 사용)

    br보다 p가 강력한 이유 : p테그 안에 CSS를 사용 가능(이 문장은 p위에 70px의 여백이 있음)


    img테그! src쪽에 이미지의 주소를 넣으면 사용 가능

    저장된 이미지의 경로를 넣어 사용할수도 있음 이때 경로 기준은 현제 html의 파일이 있는 폴더

    또한 width로 사진의 크기 조절 가능(위 사진은 width = 50%)

  • 리스트 테그 : 목차등에 사용
    • ul(unordered list)테그는 각 리스트 테그를 묶어줌
    • 두 리스트 테그의 구분을 위함
    1. ol(ordered list)테그는 왼쪽에 숫자를 만듬
    2. 리스트를 빼고 더할때 간편함

  • a(anchor) 테그
    • 링크의 기능 href(Hyperlink Reference)에 주소를 넣어 하이퍼링크로 사용 가능
